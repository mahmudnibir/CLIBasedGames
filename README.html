The Ultimate Dungeon Escape

RPG Adventure

Embark on an epic text-based dungeon crawler with rich animations, turn-based combat, and multiple room types. Battle fearsome enemies, solve puzzles, and escape with your life!

Premium Features

Rich Text Animations - Beautifully animated text with color effects
Turn-Based Combat - Strategic battles with attack/defend/item options
Puzzle Rooms - Solve riddles to progress
Loot System - Collect gold and items
Boss Battles - Epic final encounter
Random Events - Unique encounters every playthrough

Installation

pip install rich
python Dungeon.py

Controls

attack Basic attack
defend Reduce damage
item Use inventory
left/right Navigate dungeon
inventory Check items
